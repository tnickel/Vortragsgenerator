ElevenLabs & OpenRouter Kurzanleitung

1. ElevenLabs Account erstellen & API-Key finden

Melde dich kostenlos unter elevenlabs.io an. Klicke oben links auf das Dropdown-Menü (standardmäßig ElevenCreative) und wähle dort ElevenAPI aus (gelb markiert). Gehe anschließend im linken Navigationsmenü ganz unten unter Konfigurieren auf API-Schlüssel und kopiere deinen API-Key.

Screenshots für Schritt 1 anzeigen
1. Profil-Icon -> Profile + API-Keys
2. Im Menü 'ElevenAPI' auswählen
3. Ganz unten auf 'API-Schlüssel' klicken
2. Eigene Stimme klonen & Voice-ID kopieren

Um deine Stimme zu klonen:

  1. Klicke im Menü links auf Stimmen (Voices) und wechsle oben auf den Reiter Meine Stimmen.
  2. Klicke auf den Button + Stimme erstellen oben rechts.
  3. Wähle im Popup-Fenster die Option Sofortige Stimmenklonung aus.
  4. Klicke im Upload-Bereich auf Audio aufnehmen.
  5. Erlaube im Browser-Popup den Zugriff auf dein Mikrofon (Beim Besuch der Website erlauben oder Dieses Mal erlauben).
  6. Wähle im Dropdown-Menü das richtige Mikrofon aus (z. B. dein Headset oder USB-Mikrofon) und klicke auf Start, um zu sprechen.
  7. Nach dem Erstellen klicke bei deiner Klonstimme auf die drei vertikalen Punkte und wähle Stimmen-ID kopieren.
Screenshots für Schritt 2 anzeigen
1. Menü 'Stimmen' -> 'Meine Stimmen'
2. '+ Stimme erstellen' anklicken
3. 'Sofortige Stimmenklonung' wählen
4. 'Audio aufnehmen' anklicken
5. Mikrofon-Freigabe erteilen
6. Mikrofon auswählen & starten
7. Auf 3 Punkte klicken & ID kopieren
3. OpenRouter API-Key erstellen

Gehe auf openrouter.ai, erstelle einen Account und gehe in deinen Einstellungen auf Keys -> Create Key. Falls dieser Key im System leer gelassen wird, greift die Anwendung automatisch auf deine Windows-Umgebungsvariable OPENROUTER_API_KEY zurück.
